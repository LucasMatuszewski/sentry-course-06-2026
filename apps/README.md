# Demo applications

| Directory | Status | Purpose |
|---|---|---|
| `api-spring` | Required P0 | Spring Boot 4 PolicyLab API and server-side Sentry examples |
| `angular` | Required P0 | PolicyLab browser client and source-map/tracing examples |
| `android` | Source pending; AI Studio fallback available | Primary Polish mobile event laboratory |
| `android-sentry-reference` | Standalone reference | Small adaptation of the official Sentry Android sample |
| `ios-sentry-reference` | Static reference | Swift sample; requires macOS/Xcode for runtime verification |

Each application has its own README, build instructions, Sentry project mapping, and verification limitations.
