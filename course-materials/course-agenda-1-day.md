# Sentry for enterprise developers — one-day workshop

**Date:** 2026-06-30  
**Delivery:** Microsoft Teams  
**Time:** 09:00–17:00 Europe/Warsaw  
**Audience:** five senior developers/architects  
**Primary stack:** Angular, Spring Boot 4, Android/Kotlin; iOS/Swift reference

## Learning outcomes

Participants will be able to:

- Navigate from an issue to an evidence-based root-cause hypothesis.
- Configure projects, environments, releases, ownership, tags, and safe custom context.
- Distinguish source maps, JVM source context, Android mappings/native symbols, and iOS dSYMs.
- Follow an Angular/Android request through a Spring Boot trace.
- Assess release regressions and error trends.
- Design privacy controls and useful alerts without excessive noise.
