# Interactive course materials

`index.html` is generated only after the Markdown review checklist is approved.

The final presentation must:

- work without a CDN or analytics;
- use Europe/Warsaw time and the 2026-06-30 course date;
- show reminders at 11:00, 13:00, 13:30, and 15:30;
- support snooze/dismiss and an optional sound/browser notification;
- display the Sages survey link and a local QR asset at 13:30;
- provide keyboard navigation, progress, print styling, responsive Teams layout, exercise timers, glossary/source links, and trainer notes;
- expose a query-controlled clock for automated reminder testing.

Review [the trainer content gate](../manual-review-checklist.md) before generating the final HTML.
